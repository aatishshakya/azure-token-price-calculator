Assumptions & how this is calculated
  • Per-message size: ~1,000 input + 600 output tokens (a typical chat turn with some context/RAG).
  • Messages/day per person: Light ≈ 8 · Moderate ≈ 25 · Heavy ≈ 70.
  • Prices (Azure OpenAI Global Standard, pay-as-you-go, USD per 1M tokens — input / output, verified against the Azure OpenAI pricing page on 23 Jun 2026): GPT-5-nano $0.05 / $0.40 · GPT-4o-mini $0.15 / $0.60 · GPT-4.1-mini $0.40 / $1.60 · GPT-5-mini $0.25 / $2.00 · GPT-4.1 $2.00 / $8.00 · GPT-4o $2.50 / $10.00 · GPT-5.1 $1.25 / $10.00 · GPT-5.4 $2.50 / $15.00 · GPT-5.5 $5.00 / $30.00.
  • Anthropic Claude prices (Global Standard via Microsoft Foundry — a separate Marketplace-Models product, not Azure OpenAI; USD per 1M tokens, input / output): Claude Haiku 4.5 $1.00 / $5.00 · Claude Sonnet 4.5 $3.00 / $15.00 · Claude Opus 4.1 $15.00 / $75.00. Source: Microsoft Azure blog, "Introducing Anthropic's Claude models in Microsoft Foundry" (18 Nov 2025) — link. Newer Claude versions (e.g. Opus 4.6) have since launched on Foundry; re-check current rates before committing.
  • Overhead: token spend is multiplied by 1.15 (low) to 1.40 (high) for support plan, networking/Private Link, logging, and egress — the consistently-reported real-world markup. "Expected" uses ~1.25×.
  • Not included: Data Zone / Regional surcharges (~10% over Global), Priority Processing, cached-input discounts, provisioned throughput (PTU) reservations, fine-tuning, image/voice models, or the build cost of the chat interface. PTUs only pay off above ~150–200M tokens/month on a flagship model.
Rough planning estimate only — not a quote. Token rates change; verify at the Azure OpenAI pricing page before committing.
Prices: Azure OpenAI pricing page (Global Standard, USD), verified 23 Jun 2026 · Claude rates from the Microsoft Foundry / Anthropic announcement (18 Nov 2025). Overhead range: industry-reported real-world markups.
